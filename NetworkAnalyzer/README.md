Intelligent Network Traffic Analyzer & Anomaly Detector

This project is a real-time network traffic visualization tool built using Python and Streamlit.

 Overview
- Captures or analyzes network packets.
- Displays visual insights:
  - Protocol Distribution
  - Top Source/Destination IPs
  - Packet Rate over Time
- Detects anomalies (sudden spikes in packet flow).

Technologies Used
- Python
- Streamlit
- Plotly
- Pandas
- Scapy
